# mysql + node.jsでデータ取得
mysql + node.jsでMySQLに接続し,作成したテーブルからデータを取得する.

# データベース作成
node.js内で以下のようなテーブルを作成
| id  | name     | login_name    | kg        |
| --- | ---      | ---           | ---       |
|  1  | Hina Ito | itohina       | wellcomp  |
|  2  | Hoge Hoge| hogehoge | hogecomp  |
